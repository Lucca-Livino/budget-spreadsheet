# 💰 Budget Spreadsheet

A personal budget spreadsheet running directly in the browser — no backend, no dependencies, no hassle. Organize your **income**, **expenses**, and **investments** in one place and track your monthly balance in real time.

---

## 🚀 Features

* **Income** — Salary, freelance work, rent, bonuses, and more
* **Investments** — Fixed income, real estate funds, retirement plans, and others
* **Detailed expenses** by category:

  * Housing
  * Transportation
  * Healthcare
  * Food
  * Personal & Entertainment Expenses
  * Personal Care
  * Credit Card
* **Monthly Balance** — Automatic calculation with visual indicator (green / red)
* **Anchor Navigation** — Fixed header with links to each section
* **100% client-side** — Works by simply opening `index.html` in your browser
